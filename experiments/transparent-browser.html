Transparent Browser Window

Tech: Javascript, D3

Linkedin Post

I had a crazy idea to create a browser that was see through.

After a bit of brainstorming, I came up with the idea of using the svg's viewBox settings.

I set the same image in both browser windows, but as the overlaid browser moves it sends it's positional coordinates to the viewBox.

I then wanted to be able to click on the image to make a red dot and it appear, but also appear on the main browser image.

This has to be the most visually pleasing thing I have done!
