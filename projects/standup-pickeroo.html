Standup Pickeroo

Tech: Javascript

Github Visit Site

Another work project. This was designed with the sole purpose of being able to randomly pick people to talk at our standups

Vanilla Javascript, nothing fancy

I have an animation loop that (when a boolean is true, ie isSpinning) will cycle through the names of the list.

The slowing down is achieved by counting the frames through every loop of the animate function. When the remainder of the 'frames modulo variable is equal to 0' then it changes the value of that variable making the time steps greater
